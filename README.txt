Controls
--------
Movement          - WASD
Sprint            - Shift
Jump              - Spacebar
Interact          - E, Left Click
Quit/Start Screen - Esc


Configuration
-------------
When first run, Voi will generate a "voi.cfg" file for configuration of resolution, fullscreen, vertical sync and mouse sensitivity.
